راهنمای اجرای بازی السا - Sad Girl Game

فایل‌های داخل پوشه:
- game.py : فایل اصلی بازی (نیاز به Pygame دارد)
- assets/ : تصاویر بازی (پس‌زمینه، فریم‌های السا، سایه)
- sad.mp3 : در صورت وجود، پخش خواهد شد. اگر نیست برنامه تلاش می‌کند آن را از اینترنت دانلود کند.

نصب پیش‌نیاز‌ها:
1) نصب پایتون (نسخه 3.8+)
2) نصب pygame:
   pip install pygame

اجرای بازی:
- در ویندوز یا لینوکس: در همان پوشه، دستور:
   python game.py

تبدیل به فایل اجرایی (اختیاری):
- نصب pyinstaller:
   pip install pyinstaller
- ساخت تک فایل اجرایی:
   pyinstaller --onefile --add-data "assets;assets" game.py
  (در لینوکس از ':' به جای ';' استفاده کنید)

نکته درباره موسیقی:
برنامه تلاش می‌کند فایل موسیقی را از لینک داده شده دانلود کند. اگر دانلود نشد، بازی بدون موسیقی اجرا می‌شود.
